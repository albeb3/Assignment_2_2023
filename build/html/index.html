Action Client Node.

Subscribes to:

/odom Publisher to :

/posvel SimpleActionClient:

/reaching_goal

Author

Alberto Bono 3962994

Date

04/03/2024

Description:

Allows users to set a target (x, y) or cancel it. Utilizes action server feedback/status to determine target achievement. Publishes robot position and velocity as a custom message (x, y, vel_x, vel_z) based on the topic

/odom.

Service Node.

Subscribes to:

/posvel /reaching_goal/goal

Author

Alberto Bono 3962994

Date

04/03/2024

Server:

/dist_vel_service

Description:

Subscribes to robot position and velocity through a custom message. Computes the distance from the target and the robot’s average speed. Parameterized with the averaging window size for speed calculations.
